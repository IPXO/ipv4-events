ipv4.events — The IPv4 Era

From ARPANET to Dual-Stack — and Beyond

A retro-styled timeline of technological milestones across the IPv4 era: standards, operating systems, programming languages, browsers, mobile & wireless, security & policy, finance & markets, cloud, serverless, edge, CDNs, data centers, submarine cables, IXPs/NOGs, AI, space, satellite internet, and more.

About

ipv4.events is curated by IPXO and a growing community of contributors. All event data is openly maintained in our GitHub repository. Learn how to get involved in the README and Contributing Guide.
